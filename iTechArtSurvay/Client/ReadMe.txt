For testing:
1) Open the directory in console.
2) Type "webpack"
3) Open index.html or test.html
(type "wepback --watch" to auto-recompile your changes)

for testing with server: "npm start" in console and move to localhost:3000


structure:
build - the directory in wich your built (compiled) files (bundle.js, bundle.css and etc) will appear.
components - React components (jsx and js files). it should include directories for every module.
app.jsx - entry point for webpack
